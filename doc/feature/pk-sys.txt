      $HIC$“江湖”总是给人一种血腥的感觉，每天每时甚至每刻都在不断上演着各种阴
险狡诈、杀人放火的场面。所谓“人在江湖中 身不由己”，就算你只是想过平平淡
淡、与世无争的生涯，可你既然选择了江湖，那么就难免深陷其中。江湖是一个仇杀
，恩与怨的终结地，武力就是一切的说明、一切的理由。而作为一个游戏系统里的江
湖一样保留了这些特性，我们以一套游戏里的规则来描述江湖里的法则。$NOR$

$HIY$一、问：什么情况下杀人会成为杀人犯，并被通缉？$NOR$
    答：同时符合如下条件的：
        1、主动与对方战斗，并把对方杀死。
        2、对方不是杀人通缉犯。

$HIM$二、问：怎样才算正当防卫杀人？$NOR$
    答：符合下面条件的：
        1、对方主动与你战斗，而被你杀死。
        2、对方本身就已经是杀人通缉犯。
        3、杀死自己的其他 ID，也就是杀人的 ID 和被杀的 ID 在同一 IP 线上。

$HIR$三、问：杀人通缉犯都有一些什么限制或惩罚？$NOR$
    答：具体如下：
        1、 死亡损失加倍。
        2、 所有 NPC 见到你会主动杀你。
        3、 和其他杀人通缉犯见面会自动互相对杀。
        4、 和自己同 IP 线上的 ID 撞见会自动互相对杀。
        5、 无法使用 fly 指令传送。
        6、 无法使用奇门遁术。
        7、 无法使用泰山盟主、使者像的存、取功能。
        8、 无论你是否成年，其他玩家都可以直接对主动战斗或直接杀你。
        9、 会被拜师大理做官的玩家直接缉拿过去杀死。
        10、无法使用面具及其他易容术。
        11、每多杀一人次会被通缉 8～25 小时。
        12、每天杀 5人次 会被系统关进黑牢。
        13、如果你已是杀人通缉犯，再继续杀人对手不会再有什么损失。
        14、杀人犯组队分不到队伍经验。
        15、杀人犯呆在不能战斗的地方会遭天谴，每次天谴掉精神、气血上限 10%。
        16、如果你只是打晕别人，再把对方背到其他 NPC 那让 NPC杀死，或者再用
            小号把对手杀死，你一样会被认为是协助杀人而被判为杀人从犯。协助
            杀一人次会被通缉 4～12 小时。
        17、杀人犯线上所有 ID 系统会统计并公布以杜绝用小号滥杀。关于统计资料
            及杀人犯名单，可至扬州、大理城门处查看告示(look gaoshi)得知。

$HIG$四、问：我是杀人通缉犯，如何才能不被通缉呢？$NOR$
    答：解除通缉途径具体如下：
        1、 等待通缉时间过去。
        2、 死亡，每死亡一次减少通缉时间 1～4 小时。
